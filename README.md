# 🥗 Calorie Tracker

Приложение для учёта калорий с графическим интерфейсом, базой продуктов и распознаванием еды по фото через локальный ИИ.

---

## Быстрый старт

1. Скачай и распакуй проект
2. Дважды кликни на `calorie_tracker.bat`
3. Зависимости установятся автоматически, приложение запустится

---

## Требования

- Python 3.10+
- Windows (для `.bat` файла)

---

## Установка зависимостей вручную

```bash
pip install -r requirements.txt
```

---

## Запуск

```bash
python main.py
```

## Настройка ИИ (вкладка "Фото AI")

Для работы вкладки "Фото AI" нужен локальный ИИ-сервер. Модель не входит в состав проекта — её нужно установить отдельно.

### Шаг 1 — Установи LM Studio

Скачай с [lmstudio.ai](https://lmstudio.ai) и установи.

### Шаг 2 — Скачай модель

В LM Studio:
1. Перейди в раздел поиска моделей (иконка лупы)
2. Найди `moondream-2b-2025-04-14-4bit`
3. Нажми Download (~3.75 GB)

### Шаг 3 — Запусти сервер

1. Перейди в раздел **Developer → Local Server**
2. Загрузи модель `moondream-2b-2025-04-14-4bit`
3. Нажми **Start Server**
4. Убедись что сервер запущен на `http://localhost:1234`

### Шаг 4 — Используй

Открой вкладку "Фото AI" в приложении, выбери фото и нажми "Анализировать".

> Без запущенного LM Studio вкладка "Фото AI" недоступна, остальные вкладки работают в обычном режиме.

---

## Почему модель не включена в проект?

GitHub ограничивает размер файлов до 100 MB. Модель весит 3.75 GB — её невозможно загрузить в репозиторий. Поэтому каждый пользователь скачивает её самостоятельно через LM Studio.
