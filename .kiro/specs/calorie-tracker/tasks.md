# План реализации: Calorie Tracker

## Обзор

Реализация модульного Python-приложения для учёта калорий с CLI-интерфейсом. Архитектура состоит из трёх слоёв: хранилище (JSON-файлы) → трекер (бизнес-логика) → CLI. Используется dependency injection через Protocol для слабой связанности модулей.
